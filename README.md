# vue-next-admin

## 关于

替换默认的:`/admin/index/index`主体页面样式和登录页面`/admin/index/login`样式

## 使用

1. 下载安装本扩展
2. 打开设置：在[平台设置]-[后台框架]
3. 点击`index主体页面风格`下拉框，选择`vue-next-admin样式`
4. 点击`登录页面风格`下拉框，选择`vue-next-admin样式` (可选，登录页面可不换)
5. [F5]刷新浏览器生效。

## 注意事项

卸载本扩展时，要先把使用中的模板修改为其他。
